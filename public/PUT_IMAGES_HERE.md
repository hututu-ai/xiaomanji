# 把素材图放进这个 public/ 目录

应用会自动引用以下文件名（放进来即可，无需改代码）：

| 文件名                       | 用途                                  | 必需 |
| ---------------------------- | ------------------------------------- | ---- |
| `xiaoman-1.png` … `xiaoman-6.png` | 封面页小满**6 帧逐帧动画**（同尺寸透明 PNG） | 封面动画需要 |
| `xiaoman.png`                | 对话气泡旁的小满静态立绘 + 帧图缺失时的兜底 | 强烈建议 |
| `cover-ref.jpg`              | 水彩参考色图（暂未在页面引用）        | 可选 |

### 关于封面 6 帧动画
- 导出 **6 张同尺寸、同对齐**的透明 PNG，命名 `xiaoman-1.png`→`xiaoman-6.png`。
- 放进来后封面会自动预加载并循环播放（默认 8fps、loop 模式）。
- 想调播放方式：编辑 `src/pages/Cover.jsx` 里的
  `<XiaomanSprite size={260} fps={8} mode="loop" />`
  —— `mode="pingpong"` 是 1→6→1 来回（适合"不接回去"的呼吸动作），`fps` 调速度。
- **降级**：帧图没齐 → 用 `xiaoman.png` 静态显示；都没有 → 「满」水彩玉珠占位，绝不报错。

> 放完可以删掉这个说明文件。
